# Дизел Сервис Славе

Modern bilingual website for Diesel Service Slave, built with Next.js 16 and React 19.

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Production

```bash
npm run lint
npm run build
```

The project uses Next.js static export. The deployable output is generated in `out/`.
